## <a data-ajax="true" id="app" accesskey="app-h"><span class="type">#</span> Uygulama Dizini</a>
Dizin `app`,burda uygulamanızın çekirdek kısmı olabilir.

## <a data-ajax="true" id="bin" accesskey="bin-h"><span class="type">#</span> Komut Dosya Dizini</a>
Dizin `bin`,burda terminal üzerinden çalışacak dosyalar vardır.

## <a data-ajax="true" id="etc" accesskey="etc-h"><span class="type">#</span> Yapılandırma Dizini</a>
Dizin `etc`,burda projeniz,paketleriniz ve dahası için ilgili ayarları yapıyorsunuz.

## <a data-ajax="true" id="public" accesskey="public-h"><span class="type">#</span> Genel Dizini</a>
Dizin `public`,burda projenizin herkese açık olduğu dosyaların bulunduğu kısım.

## <a data-ajax="true" id="vendor" accesskey="vendor-h"><span class="type">#</span> Vendor Dizini</a>
Dizin `vendor`,projenizinde kullanılan bağımlılıklar burada duruyor.
